# ブロックレベルコンテンツ

独立したブロックとして扱われ、前後に改行が入ります。 行の全体をまとまりとして扱う要素です。

その他にも下記のような特徴があります。

- margin と padding の設定ができる。
- 高さを指定することが可能。
- 指定がない限り、親要素の幅いっぱいに広がる。

# インラインレベルコンテンツ

文章の一部として扱う要素です。 要素の前後で改行されないため、テキストの流れを妨げることなく同じ行に表示されます。

その他にも下記のような特徴があります。

- 上下の margin は適応されない。
- 高さと幅の指定が無効になる。
- コンテンツの大きさに応じて幅が決まる。

これらの特徴や違いを理解することで、CSS を効果的に適用することができるようになります。 例えば、レイアウトを柔軟に制御することが可能になり、レスポンシブデザインを実装する際にも活きるでしょう。
